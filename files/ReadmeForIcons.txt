1️⃣ КАК ЗАМЕНИТЬ ИКОНКИ КАРТОЧЕК
═══════════════════════════════════════════════════════════════════════════════

Откройте файл: index.html

Найдите нужную карточку (например, Card 1):

    <div class="card" data-id="1" data-text="Ваша машина мешает">
      <div class="card-icon">
        <!-- ВОТ ЗДЕСЬ НАХОДИТСЯ ИКОНКА -->
        <svg viewBox="0 0 80 80" width="56" height="56">
          ...SVG КОД...
        </svg>
      </div>
      <span class="card-text">Ваша машина мешает</span>
    </div>

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 ВАРИАНТЫ ЗАМЕНЫ:

┌─────────────────────────────────────────────────────────────────────────────┐
│ ВАРИАНТ A: Заменить на свой SVG код                                         │
└─────────────────────────────────────────────────────────────────────────────┘

Замените весь блок <svg>...</svg> на свой:

    <div class="card-icon">
      <svg viewBox="0 0 100 100" width="56" height="56">
        <!-- ВАШ SVG КОД ЗДЕСЬ -->
        <circle cx="50" cy="50" r="40" fill="#22c55e"/>
      </svg>
    </div>

┌─────────────────────────────────────────────────────────────────────────────┐
│ ВАРИАНТ B: Использовать эмодзи                                              │
└─────────────────────────────────────────────────────────────────────────────┘

Удалите <svg>...</svg> и вставьте эмодзи:

    <div class="card-icon" style="font-size: 48px;">
      🚗
    </div>

Другие эмодзи машин: 🚙 🚕 🚗 🚓 🚔 🏎️ 🚘 🚐 🚛

┌─────────────────────────────────────────────────────────────────────────────┐
│ ВАРИАНТ C: Использовать изображение                                         │
└─────────────────────────────────────────────────────────────────────────────┘

Замените на <img>:

    <div class="card-icon">
      <img src="images/car-blocking.png" width="56" height="56" alt="Машина мешает">
    </div>

Не забудьте положить изображение в папку images/

4️⃣ ИСПОЛЬЗОВАНИЕ FONT AWESOME (Готовые иконки)
═══════════════════════════════════════════════════════════════════════════════

1. Добавьте в <head> секцию файла index.html:

    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">

2. Используйте иконки:

┌─────────────────────────────────────────────────────────────────────────────┐
│ Пример для карточки:                                                        │
└─────────────────────────────────────────────────────────────────────────────┘

    <div class="card-icon" style="font-size: 48px; color: #3b82f6;">
      <i class="fas fa-car"></i>
    </div>

┌─────────────────────────────────────────────────────────────────────────────┐
│ Популярные иконки машин:                                                    │
└─────────────────────────────────────────────────────────────────────────────┘

    <i class="fas fa-car"></i>              <!-- Обычная машина -->
    <i class="fas fa-car-side"></i>         <!-- Машина сбоку -->
    <i class="fas fa-car-crash"></i>        <!-- Разбитая машина -->
    <i class="fas fa-ban"></i>              <!-- Знак запрета -->
    <i class="fas fa-door-open"></i>        <!-- Открытая дверь -->
    <i class="fas fa-window-maximize"></i>  <!-- Окно -->
    <i class="fas fa-lightbulb"></i>        <!-- Лампочка (фары) -->
    <i class="fas fa-exclamation-triangle"></i> <!-- Предупреждение -->
    <i class="fas fa-comments"></i>         <!-- Чат -->

🔗 Найти больше иконок: https://fontawesome.com/icons

Q: Как изменить цвет SVG иконки?
A: Измените атрибут fill="цвет" в SVG коде
   Пример: fill="#22c55e" (зеленый)

Q: Как изменить размер иконки?
A: Измените width и height:
   width="56" height="56" - для карточек
   width="44" height="44" - для логотипа

Q: Иконка не отображается, что делать?
A: Проверьте:
   1. Правильный путь к файлу изображения
   2. Файл действительно существует
   3. Нет опечаток в коде
   4. Открыта консоль браузера (F12) для ошибок

Q: Как сделать анимацию иконки?
A: Добавьте CSS в styles.css:
   
   .card-icon:hover {
     transform: scale(1.2) rotate(5deg);
     transition: 0.3s;
   }
